# Code of Conduct

## Our Pledge

This project follows the [CNCF Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md).
All participants — contributors, maintainers, and users — are expected to uphold its
standards in every project space, including GitHub issues, pull requests, discussions,
and any other communication channels associated with this repository.

## Reporting a Violation

If you witness or experience behaviour that violates the Code of Conduct, please report
it using the mechanisms described at:

**https://github.com/cncf/foundation/blob/main/code-of-conduct.md**

The CNCF Code of Conduct Committee handles reports submitted through the process
documented at that URL.

For project-specific concerns that do not rise to the level of a formal CNCF report,
you may also contact the CloudEvents project maintainers directly through the
[CloudEvents community channels](https://github.com/cloudevents/spec#community).

## Enforcement

Violations of the Code of Conduct may result in consequences as outlined in the CNCF
Code of Conduct, up to and including removal from the project community. The CloudEvents
project maintainers are responsible for enforcing these standards within the project's
spaces.
